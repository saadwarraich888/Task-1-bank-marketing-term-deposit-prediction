# Term Deposit Subscription Prediction

## Objective
Build a machine learning model to predict whether a bank customer will subscribe to a term deposit.

## Dataset
Bank Marketing Dataset (UCI Machine Learning Repository)

## Approach
1. Data preprocessing
2. Feature encoding
3. Exploratory Data Analysis
4. Model training
5. Model evaluation
6. Explainable AI using SHAP

## Models Used
- Logistic Regression
- Random Forest

## Evaluation Metrics
- Confusion Matrix
- F1-score
- ROC-AUC

## Results
Random Forest achieved the best performance.

## Insights
Customer contact duration, balance, and age strongly influence term deposit subscription.
